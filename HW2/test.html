Mobile Object Detection AI

This is a mobile app that currently implements tensorflow's coco-ssd object detection model using tensorflow in the react native expo environment. It processes live camera data from the device's camera and draws predictions onto the screen via an overlaying canvas.

HTML, CSS, JavaScript, React Native
